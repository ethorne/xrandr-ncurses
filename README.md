# Xrandr-ncurses
Gérer de manière rapide les écrans avec i3

## Dépendances
* ncurses

## Utilisation
permet de gérer les affichage vidéo:
* Position de l'affichage
* Reload la polybar (il faut changer le chemin)
* Reload nitrogen

## Installation
Il suffit de copier le `hdmi.sh` dans `~/.local/bin/`
